# CoalesceFi SDK

Official SDKs for the [CoalesceFi](https://www.coalescefi.com) lending protocol on Solana.

| Language | Directory | Package | Install |
|----------|-----------|---------|---------|
| TypeScript | [`ts/`](./ts) | `@coalescefi/sdk` | `pnpm add @coalescefi/sdk` |
| Rust | [`rust/`](./rust) | `coalescefi-sdk` | `cargo add coalescefi-sdk` |
| Python | [`python/`](./python) | `coalescefi-sdk` | `uv add coalescefi-sdk` |

Each SDK is independently installable. See the README in each directory for full documentation including configuration, usage examples, and API reference.
